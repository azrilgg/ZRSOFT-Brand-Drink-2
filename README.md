## ⚙️ Installation & Usage ⚙️

Unleash the vault locally:

```bash
# 1. Clone the repository
git clone https://github.com/azrilgg/ZRSOFT-Brand-Drink-2.git

# 2. Enter the directory
cd "ZRSOFT WEBSITE"

# 3. Install dependencies
npm install

# 4. Ignite the dev server
npm run dev
```

> **Note**: For the best development visualization, view on modern web engines with hardware acceleration enabled to experience the fluid 60fps glows and GSAP triggers appropriately.

## 🌌 Performance Architecture 🌌

- **Smart Rendering**: Framer Motion's `AnimatePresence` efficiently handles full DOM unmounts for our heavy modals.
- **Hardware Acceleration**: Extensive use of `will-change: transform`, `opacity`, and GPU-accelerated CSS rules for zero-lag mobile scrolling.
- **Dynamic Viewports**: Utilizing `100dvh` combined with safe-area spacing to guarantee flawless edge-to-edge display on iOS/Android.
